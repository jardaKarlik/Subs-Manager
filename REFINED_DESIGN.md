# Refined Design with Visual Hierarchy

## Layout Concept

### 1. HERO SECTION (Eye-catching)
- **Giant number**: $480 (monthly spend)
- **Trend chart**: Line graph showing 6-month history
- **Change indicator**: ↑ 12% vs last month
- **Size**: 72px font, gradient yellow-orange

### 2. STATS GRID (Quick insights)
- **Category chart**: Donut showing top 5 categories
- **Active count**: Progress bar visualization
- **Idle warning**: Wasted $ highlighted
- **Yearly total**: Annual projection

### 3. CATEGORY CARDS (Visual grouping)
- Emoji icons per category
- Cost per category with color
- Click to filter subscriptions

### 4. SUBSCRIPTIONS (Details)
- Grid/List view toggle
- Filter: All/Active/Idle
- Individual cards with hover effects

## Key Improvements
- Visual hierarchy: Most important = biggest
- Charts for insights (trend line + donut)
- Idle cost warning (actionable)
- Category distribution at a glance
